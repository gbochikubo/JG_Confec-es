Requisitos Funcionais

1- O usuário deverá ser capaz de se cadastrar.
	1.1- O software deve permitir o registro do nome;
	1.2- O software deve permitir o registro do e-mail;
	1.3- O software deve permitir o registro de senha;
	1.4- O software deve permitir o registro de endereço.
	1.5- O software deve perimirt o registro do cpf.

2- O usuário deverá ser capaz de realizar login.
	2.1- O software deve permitir a inserção de email;
	2.2- O software deve permitir a inserção de senha.

3- O usuário deverá ser capaz de alterar dados.
	3.1- O software deve permitir a alteração do nome;
	3.2- O software deve permitir a alteração do e-mail;
	3.3- O software deve permitir a alteração de senha;
	3.4- O software deve permitir a alteração de endereço.

4- O usuário deverá ser capaz de excluir a conta.

5- O usuário deverá ser capaz de buscar itens do catálogo.
	5.1- O software deve permitir a inserção do nome do produto a ser buscado.

6- O software deverá ser capaz de adicionar novos itens ao catálogo.

7- O software deverá ser capaz de remover itens do catálogo.

8- O software deverá ser capaz de alterar itens do catálogo.

9- O software deverá permitir que o usuário adicione itens ao carrinho.

10- O software deverá permitir que o usuário remova itens do carrinho.

11- O software deverá permitir que o usuário finalize uma compra.
	11.1- O software deve permitir a inserção do número do cartão.
	11.2- O software deve permitir a inserção do código de segurança do cartão.
	11.3- O software deve permitir a inserção da data de validade do cartão.
	11.4- O software deve permitir a inserção do nome presente no cartão.
	11.5- O software deve permitir a escolha do tipo de entrega.

12- O software deverá ser capaz de gerar um relatório de vendas de um dia específico.
	13.1- O software deve permitir a inserção da data para geração do relatório.

13- O usuário deverá ser capaz de realizar um pedido.

14- O usuário deverá ser capaz de cancelar um pedido.

15- O usuário deverá ser capaz de buscar um pedido.
